# goodxu.cn — 阿旭的个人博客

这是 [阿旭](https://www.goodxu.cn) 的个人博客源码仓库。

本博客基于优秀的开源项目 [NotionNext](https://github.com/tangly1024/NotionNext) 构建，通过 Notion 进行内容的数据管理和无缝发布。

## 🌐 访问地址
👉 **[www.goodxu.cn](https://www.goodxu.cn)**

## 💡 站点定位
本博客主要记录：
- AI 产品方向的深度思考
- LLM 底层架构学习笔记
- Vibe Coding 等 AI 开发实践经验

## 🤖 Agent 友好特性
本站点已完成面向 AI Agent 读者的友好化改造：
- **`llms.txt` 支持**：根目录提供 `llms.txt` 全站内容索引协议。
- **Agent 接待大厅**：提供隐藏入口 `/for-agents`，为 AI 提供结构化、按场景推荐的高效阅读向导。
- **原生摘要解析**：单篇文章通过隐藏 HTML 块提供 `AGENT_SUMMARY` 结构化字段，让大模型直接提取核心论点及金句。
